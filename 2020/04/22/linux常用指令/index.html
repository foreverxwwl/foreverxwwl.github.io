一、文件夹、文件命令

1. 新建文件夹

mkdir 选项 路径/文件名
mkdir -p test/test 在当前文件夹下创建test文件夹，在其中再创建test文件夹

2. 删除操作

rm -f aa.txt 强制删除aa.txt
rm -rf fileDir 强制删除fileDir文件夹和里边的所有文件

3. 复制操作

cp 选项 路径/文件名

4. 移动操作

mv /zx/soft/* 复制/zx/soft目录中的所有文件到当前目录
mv a.txt ./test/a.txt 复制当前目录a.txt到当前的test目录下
mv /zx/soft/ /tmp/soft 复制文件夹到/tmp/下，必须保证tmp是存在的文件夹

5. 解压操作

tar -zxvf tomcate-7.0.tar.gz -C /opt/module把当前目录下tomcate-7.0.tar.gz解压到/opt/module文件夹下。

二、进程控制

ps命令——查看静态的进程统计信息（Processes Statistic）

结合管道操作和grep命令进行过滤，用于查询某一个进程的信息
例如查看tomcat的进程

三、查看端口占用

netstat -tln 用来查看linux的端口使用情况
netstat -a 查看所有的服务端口（LISTEN，ESTABLISHED）

四、环境变量

vim /etc/profile 环境变量所在文件

source /etc/profile

五、防火墙配置

1. centos6配置

service iptables status 查看防火墙状态
service iptables stop 停止防火墙
service iptables start 启动防火墙
service iptables restart 重启防火墙

2. centos7配置

防火墙基本配置

systemctl status firewalld 查看防火墙状态
systemstl stop firewalld 停止防火墙,停止重启后会恢复原来状态
systemstl disable firewalld 停止防火墙,停止重启后不会恢复原来状态
systemctl start firewalld 启动防火墙，重启后会恢复原来状态
systemctl enable firewalld 启动防火墙，重启后不会恢复原来状态

防火墙端口配置

firewall-cmd --list-ports查看防火墙开放的端口
firewall-cmd --zone=public --add-port=3306/tcp --permanent开放3306端口
firewall-cmd --zone=public --remove-port=5672/tcp --permanent关闭5672端口
firewall-cmd --reload配置生效

六、vim编辑器

:set nu 显示行数

搜索

  1. 搜索已经安装的包名 rpm -qa |grep ‘包名‘ -qa代表query，a代表all ， grep(global search rgular expression(RE) and print out the line):是一种强大的文本搜索工具 。

    例如搜索已经安装的mysql rpm -qa |grep -i mysql

  2. 按照文件名搜索文件 find / -name ’文件名‘

    find / -name mysql 搜索名称为mysql的文件
