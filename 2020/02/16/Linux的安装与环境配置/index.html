centos的网络配置

这里蛀牙是配置centos的静态ip
虚拟机设置为固定IP，不管主机在什么网络环境下，是断网环境，还是连接任何网段访问外网的环境下，虚拟机的IP都固定不变，而且使用终端连接，始终不变，正常连接；
下面就是一个配置过程，本例中配置的固定ip为192.168.3.129

配置vm的NAT模式具体参数

  1. 首先在vm虚拟接中 点击编辑->虚拟网络编辑器->更改设置
  2. 选择VMnet8，照图中修改
    其中的子网ip设置时，前两段都为192.168 第三段 要和固定ip的第三段相同，第四段直接设为0；
  3. 在NAT设置中设置网关ip
    网关ip设置可参照第二步，其中第四段直接设为2（记住这个网关，在centOS中会用到）
  4. 配置笔记本主机具体VMnet8本地地址参数：
    第6步中的IP地址随意设置，但是要保证不能跟你要设置虚拟机的固定IP一样

    配置固定的ip地址

1.首先进入eth0的网络配置文件,使用vim编辑器打开


其中需要配置的是以下几个点

  1. BOOTPROTO=static 表示将ip设置为静态ip
  2. DEFROUTE=yes 表示自启动网络服务
  3. IPADDR=192.168.3.129 表示本服务器的ip地址，别的服务器通过个ip连接次服务器
  4. GATEWAY=192.168.3.2 网关：这个配置需要和vim虚拟网路适配器中的网关相同
  5. DNS1=114.114.114.114 dns1就配置这哥固定的值
  6. HWADDR=00:0c:29:3c:6c:35 真实物理地址 这个在克隆一个新的服务器时会用到

2.修改结束后退出保存。
reboot重启系统，service network restart 重启网络服务
3.最后测试配置是否成功
ifconfig查看ip

ping百度和ping windows的ip看能否ping通
