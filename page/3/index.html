hexo博客搭建

一、博客基本操作

  1. 博客部署到本地

    hexo clean 清楚缓存
    hexo g 生成静态网页
    hexo s 本地启动预览查看 http://localhost:4000

  2. 部署到git服务器上

    hexo deploy https://foreverxwwl.github.io/

  3. next主题安装

    1. git clone https://github.com/iissnan/hexo-theme-next themes/next
    2. 进入配置文件_config.yml 找到 theme 设置为 theme: next

二、next主题设置

  1. 增加标签页等页面

  1. 进入next主题的配置文件 _config.yml
  2. 找到menu: 将要添加的页面注释删掉
  3. 在blog页面 hexo n page tags 增加标签tags页面
  4. hexo n page categories 增加分类categories页面

  1. 修改头像姓名等信息

主要参考文档在主题的_config文件中修改

头像 avatar: 修改地址
主题 scheme:

设置大的标题、姓名在全局的_config.yml中的 title:

  1. 新建一篇博客

  1. hexo n hexo博客搭建
  2. 用markdowm编写博客

  1. 启用功能

（1）启用侧边栏社交链接

  1. 进入主题的_config.yml文件
  2. 将social: 一栏的注释取消

（2）启用打赏功能

  1. 进入主题的_config.yml文件
  2. 将reward项目注释删除
  3. comment：加上用语 也可在后面添加图片

（3）设置头像样式

  1. 进入目录 \blog\themes\next\source\css_common\components\sidebar
  2. 修改sidebar-author.styl文件中的样式

（3）设置首页阅读全文

  1. 在主题配置文件中找到auto_excerpt:选项
  2. 将false 改为true 并设置可看长度
  3. 也可在博客编辑时在相应位置添加
